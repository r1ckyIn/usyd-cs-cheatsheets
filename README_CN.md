## 📋 课程内容概览

### COMP2123 - 数据结构与算法
涵盖内容：
- 时间复杂度分析（Big-O 表示法）
- 排序算法（归并排序、快速排序、堆排序等）
- 图算法（BFS、DFS、Dijkstra、最小生成树）
- 树与二叉搜索树、AVL 树
- 堆与优先队列
- 哈希表
- 贪心算法
- 分治算法

### COMP2022 - 计算模型
涵盖内容：
- 正规语言与正则表达式
- 有限自动机（DFA、NFA）
- NFA 转 DFA（子集构造法）
- 泵引理
- 上下文无关语言与文法
- 乔姆斯基范式（CNF）
- CYK 算法
- 图灵机
- 可判定性与不可判定性
- 命题逻辑

### INFO1113 - 面向对象编程
涵盖内容：
- Java 基础
- Lambda 表达式与函数式接口
- 泛型与类型参数
- 集合框架
- 异常处理
- Stream API
- 文件 I/O
- JUnit 测试
- Comparable 与 Comparator

## 🚀 使用方法

1. **克隆仓库：**
   ```bash
   git clone https://github.com/YOUR_USERNAME/usyd-cs-cheatsheets.git
   ```

2. **进入课程目录：**
   ```bash
   cd usyd-cs-cheatsheets/COMP2123
   ```

3. **查看或打印 Cheatsheet** 进行考试复习！

## 🤝 贡献指南

非常欢迎大家贡献自己的 Cheatsheet！以下是贡献方式：

1. **Fork** 这个仓库
2. **创建** 新的分支用于你的贡献
3. **添加** 你的 Cheatsheet（请遵循现有格式）
4. **提交** Pull Request

### 贡献规范

- 使用清晰、易读的格式
- 文件名中包含课程代码
- 推荐使用 PNG 或 PDF 格式
- 确保内容准确且最新
- 遵守学术诚信规定
