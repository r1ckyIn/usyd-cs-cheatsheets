# Contributing to USYD CS Cheatsheets

First off, thank you for considering contributing to this repository! It's people like you that make this resource valuable for the entire USYD CS community.

## How Can I Contribute?

### Adding New Cheatsheets

1. **Fork the Repository**
   - Click the "Fork" button at the top of this repository

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/usyd-cs-cheatsheets.git
   cd usyd-cs-cheatsheets
   ```

3. **Create a New Branch**
   ```bash
   git checkout -b add-COURSEXXXX-cheatsheet
   ```

4. **Add Your Cheatsheet**
   - Create a new folder with the course code: `COURSEXXXX/`
   - Add your cheatsheet files (PNG, PDF, or Markdown)
   - Create a `README.md` in the course folder following existing format

5. **Commit Your Changes**
   ```bash
   git add .
   git commit -m "Add COURSEXXXX cheatsheet"
   ```

6. **Push and Create PR**
   ```bash
   git push origin add-COURSEXXXX-cheatsheet
   ```
   Then open a Pull Request on GitHub.

### Improving Existing Cheatsheets

- Fix errors or typos
- Add missing content
- Improve formatting or readability
- Update outdated information

### Reporting Issues

If you find errors or have suggestions:
1. Open a new Issue
2. Describe the problem clearly
3. Include course code and specific location if applicable

Before submitting, ensure your cheatsheet:
- [ ] Has accurate information
- [ ] Is well-organized and easy to read
- [ ] Covers key course topics
